Class: RaphaelRenderer

OG.renderer. RaphaelRenderer

new RaphaelRenderer(container, containerSize, backgroundColor, backgroundImage, config)

Raphael 라이브러리를 이용하여 구현한 랜더러 캔버스 클래스 - 노드에 추가되는 속성 : _type, _shape, _selected, _from, _to, _fromedge, _toedge - 노드에 저장되는 값 : shape : { geom, angle, image, text }, data : 커스텀 Object
Parameters:
Name Type Description
container HTMLElement | String 컨테이너 DOM element or ID
containerSize Array.<Number> 컨테이너 Width, Height
backgroundColor String 캔버스 배경색
backgroundImage String 캔버스 배경이미지
config Object Configuration
